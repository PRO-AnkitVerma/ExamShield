Smart Examination

Vidhyam prevents user from opening any other window while examination is in progress. This also blocks access to different shortcut keys such as cut ,copy , paste etc.. As student started giving exam from different remote locations, the supervision for these exam became difficult.

Smart Submissions

Students can submit assignments by subjects in any file format, whereas faculty can upload the definition of the assignment. Student will be able to see submissions given by faculty.

Remarking

Additionally, the oral exam (VIVA) can be marked. The feature allow faculty to give marking on assignment is also possible. Student can also see their marks.
